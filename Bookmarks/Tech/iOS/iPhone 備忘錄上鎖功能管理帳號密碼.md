# iPhone 備忘錄上鎖功能管理帳號密碼

- URL: https://www.techbang.com/posts/122845-how-to-lock-iphone-notes

- Topic: iPhone 備忘錄 lock 功能用嚟管理帳密

- Status: active

## Why I saved this

唔想裝第三方密碼管理 App 嗰陣，用 iPhone 內建備忘錄＋上鎖係最低成本方案；step-by-step 留底。

## Summary

iPhone 內建備忘錄 App 有「上鎖」功能，可以將一份「帳號密碼整理筆記」加密；解鎖用 **iPhone 密碼／自訂密碼／Face ID／Touch ID**。免裝 1Password／Bitwarden／LastPass 等第三方工具。

## Key Takeaways

### 鎖定步驟（10 步）

1. 開「備忘錄」App，揀要鎖嘅筆記（或新增）
2. 輸入內容後 → 右上角「⋯」
3. 按「鎖定」
4. 揀「使用 iPhone 密碼」或「製作密碼」→ 出現上鎖符號
5. 點入會顯示「此備忘錄已鎖定」→ 按下方「檢視備忘錄」
6. 用密碼／Face ID／Touch ID 解鎖

### 改解鎖方式

`設定 → App → 備忘錄 → 密碼` → 可改解鎖方式（密碼／FaceID／TouchID 切換）

## My Notes

**安全考量**：
- iPhone 備忘錄鎖定係**設備層** lock，唔等同 password manager 嘅 zero-knowledge encryption；同 Apple ID 同步去 iCloud 嘅鎖定備忘錄係 end-to-end encrypted（自 iOS 11.4+）。
- 對 personal 用途夠用；高敏感帳密（銀行／crypto）仍建議用 dedicated password manager（1Password/Bitwarden/Apple Passwords app 自 iOS 18 起內建）。
- **iOS 18+** Apple 已推獨立「Passwords」app（取代 iCloud Keychain UI），對純密碼管理更專業；備忘錄 lock 更適合**有額外 context／note 嘅密碼類資料**（如「銀行 X 安全提示問題答案」「特殊登入流程備忘」）。

在這個數位生活無所不在的時代，從網購平台、社群媒體到各種線上服務，幾乎每個人都需要註冊並管理許多不同的帳號與密碼。有的需要英數混合、有的要求加入特殊符號，加上各平台對密碼的要求不盡相同，即使記憶力再好，也很難不出現忘記密碼的時候。為了解決這個困擾，不妨利用 iPhone 內建備忘錄 App 的「上鎖」功能，只要建立了一份專屬的「帳號密碼整理筆記」，就能把每個電商、銀行、串流影音等各平台的登入資訊一一記錄下來，同時再設下密碼或使用 Face ID 鎖起來，確保這些資料不會輕易被其他人看到。

對於不想額外下載第三方密碼管理 App 的用戶來說，這無疑是一個省事又安全的管理方式。

### 步驟 1：第一步，開啟「備忘錄」App，並點選要鎖定的備忘錄或點選右下角「新增備忘錄」圖示。

### 步驟 2：下一步，輸入好內容後，選擇右上角「⋯」圖示。

### 步驟 3：接著按下「鎖定」。

### 步驟 4：選擇「使用 iPhone 密碼」或「製作密碼」，完成後，會出現上鎖符號。

### 步驟 5：點擊進入後會出現「此備忘錄已鎖定」的文字，點選下方「檢視備忘錄」。

### 步驟 6：即可以密碼或 Face ID 或 Touch ID 來解鎖。

### 步驟 7：另外，如要變更解鎖方式，這時需進入「設定」，接著下滑找到「App」選項。

### 步驟 8：再來點入「備忘錄」。

### 步驟 9：找到「密碼」選項後點擊進入。
