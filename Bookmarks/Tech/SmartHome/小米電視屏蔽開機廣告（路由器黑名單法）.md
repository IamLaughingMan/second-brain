# 小米電視屏蔽開機廣告（路由器黑名單法）

## Why I saved this

實用 IoT ad-block trick：唔需 root 電視、純 router 層 DNS block；對小米／其他 smart TV 通用，**可以套用到屋企任何裝置（包 Echo／Google Home／TP-Link 智能家品）**嘅 telemetry / ad 屏蔽。

## Summary

小米電視（同大多互聯網電視）開機廣告係透過 WLAN 連接 → 訪問廣告源 URL 拉取。**家用路由器後台都有「網址訪問黑名單」**，將廣告源 domain 加入即可阻斷。**完全不需 root 電視**。

> 大多數家用路由器都會有網址訪問黑名單，我們可以利用該功能，屏蔽廣告源網址。

## Key Takeaways

### 操作步驟

1. 入路由器**後台管理頁面**（一般係 192.168.1.1 或 192.168.31.1）
2. 揾「網址訪問黑名單」（不同 router 叫法唔同：URL filter／DNS filter／Parental Control／Block List）
3. 加入下面清單嘅 domain
4. 套用、重啟電視測試

### 廣告源域名清單（按重要性）

**⭐ 三條核心（一般情況下夠用）**：

```
ad.xiaomi.com
ad1.xiaomi.com
ad.mi.com
```

**🔧 完整 15 條清單**：

```
tat.pandora.xiaomi.com
upgrade.mishop.pandora.xiaomi.com
logonext.tv.kuyun.com
config.kuyun.com
api.io.mi.com
mishop.pandora.xiaomi.com
dvb.pandora.xiaomi.com
api.ad.xiaomi.com
de.pandora.xiaomi.com
data.mistat.xiaomi.com
jellyfish.pandora.xiaomi.com
gallery.pandora.xiaomi.com
o2o.api.xiaomi.com
bss.pandora.xiaomi.com
gvod.aiseejapp.atianqi.com
```

### 適用對象（推測）

- ✅ **小米電視**（直接針對）
- ✅ **其他互聯網電視**（原理同：WLAN-based ad fetch）
- ✅ 連 WiFi 嘅其他小米／米家裝置（盒子／顯示器等）—— 部分 telemetry 一齊堵
- ⚠️ 如果裝置走 4G／LTE（IoT 卡），呢招無效

## My Notes

### ⚠️ 時效性提醒（重要）

文章 **2017-06**，距今 9 年：
- 小米**可能加咗新 ad 域名**（特別係 2020+ 嘅 telemetry／OTT 廣告）
- 部分清單域名**可能已棄用**
- 屏蔽 `api.io.mi.com`／`data.mistat.xiaomi.com` 可能會**影響裝置某啲功能**（雲端同步、固件更新檢測）—— 屏蔽前知道 trade-off
- **2026 建議更新做法**：用 **Pi-hole** 或 **AdGuard Home** 跑全屋 DNS-level ad-block；社群維護嘅 blocklist 比手動 router list 更新更快（搜「pi-hole xiaomi」「adguard mi-tv blocklist」）

### 進階方案對比

| 方法 | 門檻 | 維護成本 | 強度 |
|------|------|---------|------|
| **路由器黑名單**（本文） | 低（人人路由器都有） | 高（手動更新清單） | 中（域名 only） |
| **Pi-hole**（Raspberry Pi 或 Docker） | 中（要架 server） | 低（社群 list 自動更新） | 高（DNS sinkhole + 統計） |
| **AdGuard Home**（同 Pi-hole 同類） | 中 | 低 | 高 |
| **路由器刷 OpenWrt + AdGuard plugin** | 高（要刷固件） | 低 | 高 |
| **小米電視 root + adblock 修改系統** | 極高（保固失效） | 中 | 最高 |

### Side effects 要留意

- 屏蔽 `upgrade.*` 可能令電視**唔再收 OTA 固件更新**
- 屏蔽 `data.mistat.*` 屏蔽埋 usage analytics —— 對 privacy 友好但同時影響部分「小米賬戶活動」記錄
- 屏蔽 `mishop.*` 可能令內置應用商店掛掉
- **小米可能反制**：將 ad 域名 hardcoded 用 IP 而非 DNS（呢個 trick 就會失效）；2020+ 部分小米裝置已用 DoH（DNS over HTTPS）繞過 router DNS

### 同類 trick 推廣到其他品牌

- **TCL / Hisense / 創維**：同樣 WLAN-based ad，揾其廣告源 domain
- **Roku / Amazon Fire TV**：類似策略（社群已有完整 list）
- **Samsung Tizen / LG webOS**：domain 列表不同，要自己 sniff
- **Apple TV**：通常冇開機廣告（exception），但訂閱 / Search 仍可能有 promo

### 小米電視開機廣告很煩？一招教你去除
