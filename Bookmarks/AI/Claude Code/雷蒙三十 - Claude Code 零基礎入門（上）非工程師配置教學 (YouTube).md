## Summary

台灣生產力創作者雷蒙（柚智夫妻 X 雷蒙三十，lifehacker.tw）嘅 Claude Code **零基礎入門教學**（上集，35:08）。主打「非工程師也能用」——佢親自由零幫太太柚子配置一台全新 Mac，全程實作示範 Claude Code 安裝同基礎配置。下集示範實際打造第一個 AI 工作流。

## Key Takeaways

- **定位**：Claude Code 不只寫程式，係「最能幫你幹事嘅 AI 助理」；人人可開發工具（雷小蒙 AI 分身、Discord 機器人、團隊儀表板等案例）
- **AI 應用三階段**：Chat → Cowork → Code（問答 → 協作 → 貼身團隊）
- **點解直接教終端機**：別怕；AI 時代升級路徑＝願意暫時變差先突破
- **釣魚網站警告**：Google 搜尋有陷阱，認明 `claude.ai`
- **AI 記憶搬家**：由 ChatGPT / Gemini 搬去 Claude
- **CMUX 終端機**：令終端機變好用嘅秘密武器 ＋ 多視窗同時跑多個 AI 任務
- **專案習慣**：唔好喺系統層級開 Claude Code；建專案資料夾；`cc` alias 一鍵入專案 + No Flicker 模式
- **Permission 四種模式** 點揀 ＋ **安全刪除防線**（令 `rm` 唔再係永別）
- **MCP 外掛**：畀 AI 裝超能力，串接 Gmail / Calendar / Notion
