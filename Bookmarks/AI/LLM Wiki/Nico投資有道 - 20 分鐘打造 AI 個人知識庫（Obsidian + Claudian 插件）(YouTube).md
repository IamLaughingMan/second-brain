## Summary

Nico投資有道嘅 0→1 入門影片：用 **Obsidian + Claudian 插件**打造 AI 時代「外置大腦」個人知識庫。核心論點＝調教 AI Agent 嘅關鍵係**個性化數據**，所以要把散落喺微信/備忘錄嘅資料集中入 Obsidian，再用插件接 Claude。**同 cluster 其他文章角度唔同**：呢篇用 **Claudian 社區插件**（喺 Obsidian app 內 chat），而唔係 Claude Code CLI/agent 驅動 vault。

## Key Takeaways

- **核心**：好 AI Agent ＝ 餵佢個性化數據；越個性化越懂你 → 需要一個集中嘅「外置大腦」
- **痛點**：個人數據散落微信/iOS 備忘錄；備忘錄冇插件生態、接唔到 Claude 大模型
- **方案**：遷移去 **Obsidian**（豐富插件生態）+ **Claudian 社區插件** → 喺 Obsidian 內用對話框直接同 Claude 對話（＝AI 外置大腦控制台）
- **定位**：20 分鐘 0→1 入門；配套文字教程在 `invest-nav.com`
- **角度差異（重要）**：cluster 中**少數用 Obsidian 插件（in-app chat）路線**，對比大多數用 Claude Code agent
