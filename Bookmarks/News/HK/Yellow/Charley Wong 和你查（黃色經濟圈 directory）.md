# Charley Wong 和你查（黃色經濟圈 directory）

## Summary

一個開放源碼同開放數據嘅 HK **黃色經濟圈資料庫**，搜集多間商店、創作人、專頁，識別第一身表態支持 #香港人／#自由／#民主 嘅良心商戶。網站係 directory／查詢工具，唔做篩選——用家自行按喜好辨別。

## Key Takeaways

- **規模**：截至抓取時收錄 **9136 項**資料（並持續增加）。
- **定位**：directory／查詢平台，唔評論、唔篩選；高調行動／低調暗示一律收。
- **開放授權**：
  - **原碼**：GPL-3.0-or-later
  - **數據**：ODbL-1.0
- **GitHub**：[giffon/charleywong](https://github.com/giffon/charleywong) — 數據＋原碼一齊公開，鼓勵同路人重用（要註明出處）。
- **同類比較**：屬「黃店地圖／directory」家族，但獨特之處係**雙開放**（code + data），可被其他 app／研究者 fork／import。

## My Notes

- HK 後 2019 黃色經濟圈嘅 long-tail infra：當主流 app／媒體會被 takedown，open-data／open-source 嘅 directory 係保育同 fork 嘅最後一道防線（同 [[聞庫 collection.news — 蘋果日報＋立場新聞文章存檔（HK 新聞保育）]] 同源思路）。
- 9136 entry 嘅規模算大，但**呢個 page 本身只係 landing**，目錄頁要點「睇晒完整目錄」入；之後可以考慮 ingest GitHub data dump 入 `raw/`。
- 「Charley」呢個名係 phonetic 玩「查」字（搵商戶資料 = 「查」）—— 港式語感。
