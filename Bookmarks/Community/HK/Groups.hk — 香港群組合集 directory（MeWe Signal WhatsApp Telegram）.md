# Groups.hk — 香港群組合集 directory

## Summary

HK **眾包跨平台群組 directory**，畀人按類別搜索 MeWe／Signal／WhatsApp／Telegram 群組，亦可自助提交自己嘅群組 link。屬「香港社群基礎設施」一類 catalog tool（同 [[Charley Wong 和你查（黃色經濟圈 directory）]] 同類精神，但呢個係 social/community-generic 而非 politics-aligned）。

## Key Takeaways

- **覆蓋平台**：MeWe／Signal／WhatsApp／Telegram（兼容 fragmented 嘅 HK messaging landscape）。
- **18+ 大類**（按官方 description 排）：
  - 飲食｜興趣｜購物及服務｜親子｜教育
  - 移民留學｜生活及旅遊｜地區｜運動及健康
  - 寵物及種植｜行業｜吹水｜動漫電玩｜科技
  - 傳媒及社會｜個人及工作室｜Sticker｜創業及招聘｜其他
- **Entry 資訊**：每個群顯示平台 icon、群名、簡介、成員數（如有）；點入即跳該平台 group invite。
- **提交模式**：眾包（user-submitted），無 strict curation —— 質量與群活躍度因人而異。
- **時效性**：homepage 顯示嘅 group 列會 **隨時間旋轉**（snapshot 唔代表完整 catalog；要搜尋特定 group 要用 site 嘅搜尋／分類功能）。

## Why This Matters

- HK 2020+ 後 social／messaging 散落多平台（FB 衰退、Signal／Telegram 上升、MeWe 一時冒起），**呢類 cross-platform directory 解決搵群嘅 fragmentation 問題**。
- 對比類比：歐美有 t.me 嘅 official Telegram directory 或 GroupMe／Discord server lists；HK 因有獨特平台 mix（含 MeWe／Signal）需要本地化 directory。
- 對 marketer／業者：可掃 entry 估算 HK 各垂直市場嘅 community size／活躍度（粗略 signal）。

## My Notes

- **歸類爭議**：曾在 News/HK／Culture/HK／Tech/Messaging／Community/HK 四個 option 之間，使用者揀 **新 top-level `Community/HK/`** —— 純社群類獨立 root，將來收 HK 群組／論壇／社群平台等。
- **檔內 snapshot 取頭 ~40 entry**（非全 462 行）—— 因 homepage 列表動態旋轉，全 archive 反而 noise；snapshot 用作 link-rot 防護＋一時 baseline 之用。**價值在條 site／directory tool 本身**，唔在 snapshot 內容。
- 攞特定 group：直接上 https://groups.hk 用分類／搜尋；本檔唔用嚟代替 live lookup。
- 唔包含 Discord／Slack／LINE／X DMs 等其他社群平台。
