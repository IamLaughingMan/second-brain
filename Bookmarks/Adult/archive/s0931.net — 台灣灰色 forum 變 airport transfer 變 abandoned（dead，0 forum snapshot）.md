## Key Forensic Observations

1. **Forum 子頁面 0 Wayback snapshot**：
   - `forums.cgi?forum=N` 系列 URL 全部冇 archive（CDX prefix query 返 0）
   - 即使存在過、舊內容已 **永久 lost**——Wayback 無 backup
   - 用戶 supply 嘅 URL 入面有「`cgi-bin/cgi-bin/`」double-prefix，應該係 typo（CGI 慣常單一 prefix）；真實 path 應為 `/cgi-bin/forums.cgi?forum=8`
2. **Domain 命名 signature**（早期身份）：
   - `s0931` = 台灣 0931 mobile prefix（老司機式 naming）
   - cgi-bin forum + image-only landing + 無語意 subdir（`ttt/msdd/555/SPA`）→ 早期成人 forum routine
   - 「寶山綠營眷協會」掛名 = 退伍軍人團體名義承租（Front／cover？或借殼合法化）
3. **Domain 轉手痕跡**：

   - 2021 一次大改、變 6KB 含完整 service info
