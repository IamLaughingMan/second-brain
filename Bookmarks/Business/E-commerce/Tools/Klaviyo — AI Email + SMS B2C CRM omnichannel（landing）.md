## Summary

Klaviyo 係 e-commerce email/SMS marketing 嘅事實標準（market leader），2023 年 IPO 上 NYSE。Landing page 重點：
- **196,000+ 商戶 / 100 國家**（Privy 10K / Zotabox 10K 嘅 ~19x 規模）
- **350+ pre-built integrations**（vs 競品數十個）
- **AI agents 矩陣**：
  - **Composer**：URL → 整套 on-brand campaign
  - **Marketing Agent**：自動學 site + 起 campaigns / flows / forms
  - **Customer Agent**：autonomously resolve **65%** support questions + 推薦 + converts
  - **K:AI Agents**：跨 channel decisioning
- **Omnichannel**：Email / SMS / RCS / WhatsApp / mobile push / web events 統一 customer profile
- **AI ecosystem 整合**：Klaviyo ChatGPT app、**Klaviyo Claude Connector**、**Klaviyo MCP**（agentic workflows）— 罕見強嘅 LLM-side support
- **Pricing**：Free plan available（core features），paid 按 contacts + channel usage scale，Enterprise dedicated CSM + SLA

## Key Takeaways

1. **vs Privy / Zotabox（核心對比表）**：

   | | Zotabox | Privy | Klaviyo |
   |---|---------|-------|---------|
   | 客戶數 | 10K | 10K | **196K** |
   | Focus | 20+ widgets | Email + SMS + Popup | **B2C CRM + AI** |
   | 平台 | 多 (Shopify+9) | 主 Shopify | 多 (Shopify primary) |
   | AI agents | 無 | 弱 | **強（K:AI / Composer）** |
   | LLM connector | 無 | 無 | **Claude / ChatGPT / MCP** |
   | Channels | popup/bar | email/SMS | email/SMS/RCS/WhatsApp/push |
   | Free tier | trial only | 15-day trial | **真 free plan** |
   | Enterprise | 無 | 弱 | **dedicated CSM + SLA** |
   | IPO | private | private | **NYSE: KVYO** |

2. **AI agents 係 2025-2026 嘅 narrative**：Composer 由 URL 一鍵起 campaign + Marketing Agent 自動 build flows — 大幅降 setup time（Privy 5-min setup vs Klaviyo「URL → done in minutes」）。但 reality：autonomous agent 嘅 quality 仍係 case-by-case。
3. **Customer Agent 65% autonomous resolution**：landing claim，唯 Klaviyo 內部 metric；真實 deflection rate 跨 vertical 差異大、要 verify。
4. **MCP + Claude Connector = LLM developer 友好**：對 user（用緊 Claude Code）有 strategic value — Klaviyo 數據可直接接落 LLM agent，做 cross-tool automation（如 idea：Claude 讀 Klaviyo flows → 對比 best practice → 自動生 A/B variants）。**Privy / Zotabox 冇等價產品**。
5. **Free plan**：Klaviyo 唯一提供真 free plan（up to ~250 contacts／500 emails/月）— 對 [[自家品牌金屬錶微創業（LIHKG 2022 帖啟發）|month 0 testing]] 適合到瓜，**比 Privy 15-day trial 更友好**。
6. **Cost trajectory**：Free → $20/mo (500 contacts) → $45/mo (1K) → $150/mo (5K) → $700+/mo (15K+)。每加 1K contacts 約 $20-30/mo 增量。SMS 另計（每條 ~$0.01-0.03 US）。

## My Notes

- **三 tier stack 嘅 default progression**：
  - **Month 0-3（pre-traction）**：Klaviyo free plan（250 contacts）+ Shopify 自家 popup → 試水
  - **Month 3-12（$5K-100K USD GMV）**：跳 Privy + CSM coaching 或繼續 Klaviyo paid（$20-150/mo）— 視乎需唔需要人 hand-holding
  - **Month 12+（$50K+ USD GMV）**：Klaviyo full suite + AI agents 解放精力做 product / acquisition
- **2026 觀察**：Klaviyo 2024-2025 push AI agents 同 Composer 係明顯 platform strategy — 由「email + SMS tool」轉「B2C CRM platform」，long-term ARR + retention 上揚。Privy 仍企硬 SMB email/SMS 利基，但 risk 係 Klaviyo free plan + AI agents 吸晒 entry-level customers。
- **Klaviyo Claude Connector / MCP 應深入研究**：對 user own workflow potentially 有 leverage（例：[[Personal/Ideas|搵錢 ideas]] 任何 e-commerce play，Klaviyo data ↔ Claude agent 嘅 integration 可以做 next-level automation）。值得做 dedicated bookmark / wiki page 研究。
- **HK e-commerce 客角度**：HK Shopify 中型店（HK$50K-500K GMV/月）幾乎 default = Klaviyo（中文 community 推薦壓倒）；Privy 喺台灣社群仲有市場、HK 比較少見。
- **IPO + public 透明度**：作為上市公司，KVYO 嘅 quarterly earnings / SEC filing 可拎做 reality check（vs Zotabox / Privy 全黑盒）。2026 Q3 net retention rate / ARR growth 可推敲產品健康度。
- **Lock-in 風險最高**：350 integrations + AI agents + CDP = data gravity 最大；遷去 Privy / Zotabox 等於放棄 AI + integration 生態。

### FAQ excerpts（landing 列出）

**What is Klaviyo?**
> Klaviyo is the autonomous B2C CRM and AI marketing platform that unifies customer data and automates email, SMS, RCS, WhatsApp, and mobile push to drive personalized, revenue-generating campaigns. Built as both a system of record and system of action, Klaviyo combines a customer data platform, real-time signals, and AI agents to power automated insights and one-click execution.

**How much does Klaviyo cost? Is there a free plan?**
> Yes. Klaviyo offers a free plan with core features, and paid plans scale based on the number of contacts and channel usage (such as email and SMS). Enterprise pricing is also available.

**What integrations does Klaviyo offer?**
> Klaviyo offers native integrations with ecommerce platforms like Shopify, plus connectors for payment providers, CRMs, analytics tools, and messaging platforms. We also integrate with your favorite AI tools with our **Klaviyo ChatGPT app, Klaviyo Claude Connector, and Klaviyo MCP** for building agentic workflows. **What channels does Klaviyo support?**
> Klaviyo unifies email, SMS / Text, RCS, mobile push, WhatsApp and web events into a single customer profile so you can run consistent, data-driven omnichannel marketing.

**How does Klaviyo support enterprise brands?**
> For enterprise brands, Klaviyo is the unified B2C CRM built to replace fragmented legacy stacks — consolidating data, execution, and analytics into one secure, scalable platform. Klaviyo deploys rapidly — with enterprise-grade security, audit-ready compliance, dedicated CSMs, and peak-season uptime backed by SLAs.
