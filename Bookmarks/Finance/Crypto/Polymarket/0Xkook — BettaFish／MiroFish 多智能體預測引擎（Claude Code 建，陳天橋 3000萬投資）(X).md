# BettaFish／MiroFish 多智能體預測引擎（Claude Code 建，陳天橋 3000 萬投資）

> [!info] Bookmark · ai-agent domain · X（@HuaXi46802 0Xkook）
> **⚠️ 唔係 Polymarket 本身**——係一個多智能體**預測引擎**（MiroFish）＋其舆情前作（BettaFish）。放喺呢個 cluster 係因為佢正正係 [[Polymarket LLM-reasoning bot（叫 Claude 分析落注 · Robot Traders）]]（路線 C）嘅**底層工具角度**：多-agent 模擬 + 預測。要嘅話可搬去 `Bookmarks/AI/`。

## Summary
0Xkook 轉述一個熱門故事：中國大四學生 **郭航江（BaiFu）** 用 **Vibe Coding（Claude Code 等 AI 工具）10 日**，把大學幾年功底整合成開源項目 **BettaFish**——一個**多智能體輿情分析**助手（自動掃 30+ 中外社媒平台、數百萬留言）。BettaFish 登上 GitHub 全球 trending（~2.3 萬星），獲盛大集團創辦人**陳天橋 3000 萬人民幣**投資。後續作 **MiroFish**：新一代**多智能體 AI 預測引擎**——由現實世界抽 seed（突發新聞、政策草案、金融訊號）自動建「高保真平行數字世界」模擬去預測，再度登上 GitHub trending。

## Key Takeaways
- **Vibe Coding / Claude Code**：10 日速成，正正示範 AI-assisted 開發嘅威力（同你 vault 主題呼應）。
- **MiroFish = 多-agent 預測引擎**：seed（news/policy/financial）→ 平行世界模擬 → 預測。＝ 路線 C「模擬 + LLM 判斷」嘅通用化框架。
- 商業驗證：陳天橋（盛大）3000 萬投資 → 多-agent 預測方向有資本背書。
