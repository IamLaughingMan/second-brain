## Key Takeaways

1. **AIC 定義**：Augmentative Interspecies Communication — 跨物種輔助溝通；用 recordable button（按一下播一個錄音字）俾動物表達需求／意圖。
2. **科學基礎**：speech-language pathologist Christina Hunger 2019 起做 Stella case study、Leo Trottier 同 UCSD 合作起 TheyCanTalk longitudinal study（>10K 寵物參與）。屬 emerging field、未有 peer-reviewed conclusive evidence、爭議仍在。
3. **產品 vs 社群**：FluentPet 既賣產品（HexTile + button + accessories）又運社群 + 課程，類似 GoPro／Peloton 嘅「product + content community flywheel」。
4. **入門資源全免費**（FP 101），lock-in 點喺 Academy membership 同 live coaching；社群 forum 本身免費 read。
5. **HK / 華語適配**：暫無 HK／中文 official 分區；按鈕本身語言 agnostic（你錄乜佢就播乜），可直接錄廣東話。

## My Notes

- **使用情境**：如果養狗／貓想試 AIC button training，FluentPet 101（free 8 課）係**最低成本入門**——唔買產品都可以學 framework、用任何 recordable button 開始。
- **產品 cost reality check**（2026）：
  - FluentPet Get Started Kit ≈ US$60-80（6 buttons + 1 HexTile + manual）
  - Expansion Kits US$30-50（add buttons + tiles）
  - 對比 generic Amazon button（CGoldenWall／Learning Resources）US$10-15 — FluentPet premium 主要係 HexTile spatial system + community
- **科學成熟度警告**：
  - **TheyCanTalk** 2020 起做、係 enrolment-based observational study，**未有 peer-reviewed 結論**確認動物真係理解 abstract concepts vs 純 operant conditioning
  - Comparative animal cognition community（如 Clive Wynne、Alexandra Horowitz）對 button data 有保留：可能係 cuing artifact、selective video editing
  - 但 anecdotal evidence rich + 教學 framework 對 owner-pet bonding 仍有 net positive 即使「真係識溝通」未證實
- **同 dataset / 興趣 area**：
  - 可同既有 [[Bookmarks/Animals/Rescue/HK Pigeon Dove Rescue — 雀鳥急救指南]] 一齊歸 `Animals/` domain
  - 未來 add：Bunny / Stella TikTok account、Christina Hunger 著作《How Stella Learned to Talk》、TheyCanTalk paper
- **技術 note**：呢個 site **Cloudflare anti-bot 保護**——defuddle 403、curl UA 都 hit Cloudflare challenge wall、要 Playwright headed Chrome 過。**加入 known-sites table**：FluentPet ✅ Playwright works（同 Penana / Mobile01 同一 class）。

> **👋 Welcome to the FluentPet Community!**
>
> This is where we help each other teach our learners to communicate using soundboard AIC (Augmentative Interspecies Communication) devices.
