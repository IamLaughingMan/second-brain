## Why I saved this

雖然 b-mobile 方法已死，但**「人在台灣／港」追日本現場演出嘅 authentication 問題**到 2026 仍存在；呢篇係該問題嘅 baseline analysis、保留作 reference + 留意有冇 working successor 方法。

## Summary

「**人在台灣**」要喺日本演唱會購票網站（**CN（Creative Man）、BGD、l-tike、Pia、CNPlayguide** 等）通過會員認證有兩道牆：

1. **簡訊認證**（容易，B Mobile 等 SMS-only SIM 可破）
2. **通話認證**（難，網站會彈視窗要 user 限時撥指定號碼，掛斷後完成驗證；接通即過）

作者 kenny04s 嘅成功 protocol（**現已失效**）：

**B Mobile SIM（070 開頭）+ 國際轉運送來台灣 + 退役舊機 + 全新 Google/FB/Line 帳號 + Line Out 撥日本號**

## Key Takeaways

### 為何「通話認證」咁難破

| 替代方案 | 點解唔得 |
|---------|---------|
| **本地電信申請日本號** | 需 **在留卡** —— 觀光客冇 |
| **B Mobile**（簡訊+流量） | 過 SMS 認證 OK，**通話功能冇** |
| **SMARTalk（050 網路電話）** | 自己又要電話認證才註冊 |
| **Skype 撥日本** | 作者試過失敗 |
| **朋友借號** | 需要長期在日朋友、唔 scalable |

### kenny04s 嘅破解組合（已失效但機制清晰）

1. **B Mobile SIM**（台灣訂、開頭 070）→ 由 b-mobile 喺日本完成首次開通、再國際轉運寄到台灣
   - 070 比 050 泛用（部分網站明文 reject 050）
2. **退役舊機**（蝴蝶 2）插 B Mobile SIM
3. **新 Google／FB／Line 帳號**（避免 region 衝突）
4. **VPN 跨區下載日本 LINE**（為攞 Line Out 功能 ─ **「台灣冇」**）
5. **Line Out** 撥日本指定號碼接通 → 認證 PASS

### 認證通過後嘅後續障礙

- ❌ **海外信用卡刷不過**（多本日本網站 reject 海外 issuer）
- ❌ **商品唔送轉運**（部分商店規避）
- ❌ **票不能當天日本便利商店取**（部分要早 N 日取）

→ **過認證 ≠ 一定買到票 + 拿到票**

### Q&A 精華

- **Q：以前點做？** A：050 網路電話（SMARTalk／Skype），但「現在連 050 都要認證」
- **Q：申請完會員會唔會失效？** A：可能（網站 anti-海外機制持續加碼）
- **Q：抽選／購票／二次抽選點做？** A：作者預告續篇（**主要 / 抽選 / 購票 / 二次抽選 / 購物心得**）

### 申請會員的關鍵問題

**Q：為什麼申請會員需要這麼煩惱呢??**

A：因為現在會員大多數已經不能只填電話這麼簡單，還需要能夠**通話的功能**來做驗證。以常見的 **CN** 為例（BGD 常用）：註冊的當下會出現視窗要求你在時間內（通常是兩三分鐘）撥打上面的電話（打去不會有任何聲音一旦接通就結束了）。你連反應都來不及接著電腦上的會員申請就過了！失敗的情況就是打不通然後時間到就必須重新再驗證。

**Q：為什麼不能在台灣申請「能通話」的日本電話呢?**

A：
1. **第一次開通作業**：上一篇有提到的 B Mobile 教學只有簡訊及網路功能（這種能過最簡單的簡訊認證不能過通話認證）。但「人在台灣」的情況下只推薦此種方式，原因是大多數申請的電話都需要在日本當地「進行第一次開通」的作業，而 B Mobile 會幫你「開通好」，所以可以藉由 轉運 直接送到台灣使用。
2. **需有在留卡才能申請**：這也是最大的阻礙。至於 在留卡 是什麼我就不貼了請自行谷哥。

**Q：那之前的人是用什麼方法通過認證的?**

A：以前最常見的方式是「網路電話」（050 開頭的電話號碼），像是 SMARTalk、Skype 等等。但前者現在也要電話認證，後者我失敗，所以改尋找其他方法。

### 實際操作

首先我拿一支空機（早已退役的蝴蝶 2），直接使用 **B Mobile 的 SIM 卡（開頭是 070）**。

這裡要注意我和文章有些差異：

> **最新更新**：網友回覆其實不需掛 VPN 但需創新的 LINE 等等帳號。不掛 VPN 這點畢竟非本人實測還請各位自行嘗試!!

因為我不是用既有的台灣 SIM 卡做跨區的動作，所以在第一次使用時所有的帳號都要重創一個，像是 Google、FB、Line 等等會用到的東西。

另外當時怕失敗所以前面步驟是照著文章教學：**關 GPS 掛 VPN 用 FB 登錄 Line**。所以不曉得不照做是否能夠成功，因為我一次就成功了....

注意一點：手機的 VPN 程式，我沒有找到好用的而且大多免費的有時間限制。所以我的這個作法在申請各種帳號時就會花掉很多時間，這點要請各位自行解決!!

接著確定手機上的 Line 有了 Line Out 就算成功一半了。接著就是 會員申請 的部分，可以照著文章操作（該大大有分享其他網站+詳細解說）。
