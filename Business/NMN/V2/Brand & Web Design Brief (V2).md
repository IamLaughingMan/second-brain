# NMN — Brand & Web Design Brief

> Shopify 品牌產品銷售網站。**品牌與設計方向 + 上線前合規關卡。**
> V2(2026-06-27,effort xhigh 重做):核心策略不變(clinical-transparent spine / Direction A),**強化點 = 法規以 2025-12 FDA 實況收緊、claims-safe 教育改用真 citation、定價對標用真競品 $/g**。V1 存檔 → [[Brand & Web Design Brief (V1)]]。

## 0. 已鎖定輸入

- **目標市場**:美國 / 北美 ✅(法規 timing 啱 — 見 §9)
- **定位**:大眾平價 → **masstige / decoded-commodity**(平價但 presentation 高可信)
- **品牌**:name + logo「HYGIEIAS」(深藍 wordmark)已有;**無攝影**
- **商業模式**:先 2 隻 NMN(single hero category, 2 variant:capsule + powder)→ 之後加 products
- **策略 spine**:**Clinical Transparency / Decoded Commodity**(The Ordinary / Deciem 模式;NMN 競品參考 DoNotAge / Toniiq / Double Wood / ProHealth / Renue)

## 1. 產品定位 Positioning

**一句:** Lab-grade NMN,把成分、劑量、第三方檢測攤喺枱面,用公道價賣 —— 唔幫你嘅 marketing 埋單。

- **類目:** NMN longevity / healthspan 補充品,美國,大眾可負擔。
- **差異化:** 市場兩極 —— (a) 誇大「逆齡神話」(claim 風險 + 失信),(b) 高價扮 premium。**空檔 = 誠實、劑量清楚、COA 行先、定價公道、當客人係聰明人。**
- **競爭軸:** 透明度(純度 % + 可下載 COA)、$/gram、無廢話語氣 —— **唔係** luxury,**唔係**功效 hype。
- **品牌人格:** 像一個唔兜圈、識嘢、唔賣弄嘅藥劑師朋友。

## 2. 目標客人 Target Customer

- **主力:** 35–60 歲、美國、注重 healthspan 但價格敏感。聽過 NMN(Sinclair / podcast / Bryan Johnson),想試但唔想畀 USD 50–115 一樽溢價(真競品區間,見 §9b)。會做少少 research、會比價、有少少懷疑。
- **次要:** 由 premium brand 流失嘅現有 NMN 用家(因為貴);睇 $/gram 同純度嘅 stack 優化者。
- **唔係:** 輕奢 wellness 買家;想要臨床指導 / 被服侍嘅人。
- **心態:** 「我要真貨、第三方檢測過、公道價,唔好對住我做 marketing。」

## 3. 客人主要痛點 Pains

- NMN 資訊混亂:劑量?純度?A 牌等唔等於 B 牌?
- Premium NMN 感覺溢價 —— 懷疑係幫 marketing 埋單。
- 對行業不信:filler、underdose、無證據嘅「99% pure」。
- 美國特有:「NMN 仲合唔合法 / 買唔買到?」(2022 FDA ban → 2025 逆轉,市場仲有 noise)。
- 怕畀錢買到無效 / 假貨。

## 4. 購買前疑慮 Pre-purchase Doubts

- 純度幾多 %?有冇真 COA(邊間 lab)?
- 每粒劑量幾多?夠唔夠?(研究劑量區間見 §11)
- 係咪 fly-by-night 牌子?
- 收款安全?會唔會發貨?退貨點?
- **「點解你平過 [premium 牌]?係咪 quality 差?」** ← 必須主動拆(平係因為無 marketing 溢價,純度同等或更高)。
- 第三方檢測冇?
- (法律焦慮)喺美國買 NMN 得唔得?→ 2025 已合法,FAQ 直接答。

## 5. 網站要建立嘅信任 Trust to Build

全部要 **真**,唔用假嘢:

- **真第三方 COA**,可見 + 可下載(唔係 badge,係實際 PDF + lab 名 + batch)。
- **純度 %** + 檢測方法 / lab。
- **透明劑量:** mg/serving、servings/bottle、**$/gram**。
- **「點解我哋平」誠實解說**(反溢價敘事)。
- **創辦人 / 公司透明**(名、產地、邊個檢測)。
- **慷慨退款 / 保證**(60 日)。
- **claims-safe 教育**:中性引用**真**研究,**唔講**「逆轉衰老」;**坦白講證據仲喺 emerging 階段**(見 §10)。
- **只用真 review** + cold-start 計劃(verified-buyer、誠實 seed、顯示評價數增長而非作假)。
- **真 trust signal:** 收款安全、發貨清楚、cGMP 廠、**NDI-compliant 原料**(§9 = 強差異化)。

## 6–7. 三個 Visual Direction（含 色彩/字體/留白/圖片/CTA/section rhythm）

### Direction A — 「Lab Sheet / 臨床透明」★ 推薦
- **概念:** 產品當成 lab spec sheet。資訊本身就係美學;極簡 = 信任訊號。
- **色彩:** 近白紙底(#FAFAF8)、墨黑字、**一個**克制 accent(臨床藍 #1B2A4A,取自 logo)+ 單一安全橙作 CTA。零 gradient。
- **字體:** 精準 grotesk(Söhne / Helvetica Now / 緊版 Inter)做 body;**mono** 做數據(mg / 純度 / $)。字體做主角。
- **留白:** 密而有序;大量 hairline rule、表格、spec block。係「結構化留白」唔係 luxury 空曠。
- **圖片:** product-on-seamless、top-down、證據式;粉末/膠囊微距;**COA 本身當主視覺**。無 model、無 lifestyle、無 stock → **完美配合無攝影 + 平**。
- **CTA:** 純矩形高對比按鈕,字面文案(「Add to cart — $X / 30 days」)。無發光。
- **Section rhythm:** spec → proof(COA)→ 劑量數學 → 定價誠實 → reviews → FAQ。似 datasheet 向下捲。
- **風險:** 易冷冰;靠**文案有人味**補(語氣借 Direction B)。

### Direction B — 「Honest Utility / 現代藥房」
- 親和、白話、日常健康;比 A 暖,仍無 hype。暖中性底、muted accent、humanist sans + 安靜 serif 標題。產品在手、自然光、極少 prop。
- **風險:** 暖易飄向 generic wellness /「AI 養生站」陷阱 —— **最接近 template 雷**,要紀律。

### Direction C — 「Editorial / Field Notes」
- 品牌 = 識嘢、有立場嘅編輯。強 typographic hierarchy、雜誌排版、**footnote(引用做註腳 = 信任 + claims-safe)**、圖解/插畫(分子、劑量圖表)減低攝影依賴。
- **風險:** 易重閱讀輕購買;要 sticky add-to-cart + PDP 紀律。

## 8. 推薦 + 原因 → **Direction A（語氣借 B,教育段借 C 嘅 footnote)**

對「美國 + 大眾平價 NMN + 透明 + 唔誇大 + 無攝影 + high-conversion 無 dark pattern + 唔似 template」:

1. **形式 = 策略:** 透明度就係你嘅差異化;datasheet 美學令 value prop 變成可見(見到 spec、COA、$/gram)。
2. **天生 claims-safe:** spec/datasheet 美學自然展示事實(純度/劑量/lab)而非療效承諾 → **美國 FTC/FDA 文案風險最低**。
3. **攝影負擔最低:** product-on-seamless + COA 當圖,無需 model/lifestyle/stock → 配合無攝影 + 平。
4. **最難被誤認 template / AI 站:** 極簡 + 數據密度 = gradient/glassmorphism/SaaS 嘅相反;The Ordinary 證明咗大眾價都可以讀成「認真真牌子」。
5. **靠清晰 conversion 唔靠脅迫:** 劑量數學 + 定價誠實 + 可見 COA,說服懷疑型 value buyer,唔使 urgency / 假 review。

- **唔揀 B 做主:** 暖 wellness 最接近 generic-AI 陷阱,溝淡「decoded/honest」鋒芒 → 暖只留喺語氣。
- **唔揀 C 做主:** editorial 靚但大眾平價 + 重複購買 NMN 要快、可掃、PDP 緊 → 把 C 嘅 footnote-citation 手法用入 A 嘅教育段。

## 9. 法規現狀 + Gating action items（2025-12 FDA 實況更新）

**美國法規已逆轉,而家有官方文件坐實(好消息):**
- **2025-09-29** FDA 回應 citizen petition,裁定 **NMN 不再被剔出 dietary supplement 定義**(推翻 2022 drug-exclusion;NPA 訴訟 + ANH petition 促成)。
- **2025-12-02** FDA 去信 **SyncoZymes**,**reinstate 其 NMN NDI 認可(NDI No. 1247,原 2022-05 first NMN NDI)**;**2025-12-09** FDA 喺官網公開對 **SyncoZymes 同 Kingdomway(NDI No. 1259)** 嘅回信。→ NMN 已重返 Amazon,收款 / marketplace 風險大降。
- EU 仍 Novel Food 審批中(未通過);中國禁 NMN 入食品;澳洲另有審查。→ **美國係而家少數正式開放嘅大市場,timing 啱。**

**🔑 關卡 = NDIN 合規(取代舊「賣唔賣得」問題):**
NMN 仍屬 NDI(New Dietary Ingredient / 新膳食成分),要合法賣須被 accepted NDIN 覆蓋:
- ① **(易,推薦)** 向**已有 accepted/reinstated NDIN 嘅原料商入貨** —— **SyncoZymes(NDI #1247)** 或 **Kingdomway(NDI #1259)** 係而家有公開 FDA 回信嗰兩間 → **唔使自己 file**。
- ② **(難)** 向無 NDIN 供應商入貨 → 要自己 file NDIN(錢 + 時間 + 不確定)。

→ **直接綁 sourcing:供應商「有冇 accepted NDIN / 係咪 SyncoZymes·Kingdomway 鏈」= 硬性篩選。** `To confirm`:現有原料(G1/G3)實際出自邊間、有冇覆蓋喺 #1247 / #1259。

### 9b. 真競品定價 landscape（2026 web check,驅動 §定位)

branded NMN 普遍 **$0.6–$3.9 / g**(視乎 pack / 訂閱):
- Renue By Science SCL 250mg ≈ **$1.26/g**;Neurogan NMN 500 ≈ **$1.07/g**(sub);Neurogan NMN 900 ≈ **$0.59/g**(sub);ProHealth NMN Pro(premium 線)≈ **$1.2–3.9/g**;Toniiq 300mg×60(98% 純度,COA on-request)≈ **$1.7–2/g**。
> HYGIEIAS:capsule 300mg×60 = **$1.94/g**(mid-pack,但**贏喺 COA 攤出嚟 + ≥99%**);powder 100g = **$0.45/g**(**殺多數 branded 對手**)。定價對標表用呢個。

**Gating action items:**
1. **Sourcing 篩 NDIN:** 確認原料覆蓋喺 accepted NDIN(SyncoZymes #1247 / Kingdomway #1259)否則要自 file。
2. **DSHEA 合規:** structure/function claim + disclaimer、**禁 disease claim**、cGMP、labeling。(claims-safe datasheet 設計正好對應。)
3. **Payment:** 確認 Shopify Payments / processor 接受 supplement(風險已降,仍要 confirm)。
4. **2 隻 NMN 差異**(劑量/容量/劑型)→ 影響 PDP variant 設計。
5. **真 assets:** COA(lab 名 + batch)/ cGMP cert + 廠名 / 創辦人故事。

## 10. Claims-safe 教育素材（V2 新增 — 用真 citation,坦白講 emerging)

PDP/About 教育段中性引用,**唔講療效**:
- **NMN 係 NAD⁺ precursor**,NAD⁺ 參與細胞能量代謝 —— 屬基礎生化(可中性陳述)。
- **安全/耐受:** Fukamizu et al., *Scientific Reports* (2022) —— 健康成人口服 β-NMN **1250 mg/日,4 週,安全且耐受良好**。
- **人體證據仍 emerging:** Wen et al., *Cureus* (2024) 系統性回顧(10 RCT、437 人)—— physical performance **有改善但多數未達統計顯著**,安全性良好。**呢個 honest 框架 = 品牌資產**(誠實過 over-claim)。
> 文案範式:「We share the science so you can decide — not because we're making a promise.」上線前過 DSHEA + FTC review。

## 11. 劑量決定（V2 — open question 收尾)

研究劑量區間 **250–1000 mg/日**,250mg 最多研究;**300 mg/日有對應人體試驗**(2022,66 健康成人,60 日,改善步行表現)→ **capsule = 1 粒 300mg/日 = 60 日**,有研究依據、非任意。powder 500mg/服對應「30–60 歲 ~500mg/日」常見建議。
