# HYGIEIAS v3 — Company Build (Executive Summary)

> 由 0 重建（2026-06-28，ultracode 多 agent workflow）。把定位由「NMN 品牌」升做「**補充品公司**，NMN = 第一隻產品」。Judge-panel 贏家 = **Transparency-Max**。此頁係 v3 索引；深度見各 deliverable。

## What HYGIEIAS is now
一間 **US 補充品公司（branded house）**，用 The Ordinary / Deciem 語域行 **decoded-commodity house** —— 冷靜、字面、lab-sheet、反 hype、當客人係聰明人。**透明唔係賣點，係公司嘅 operating system。** 兩條線：**Foundations**（日常基礎）＋ **Longevity**（healthspan / NAD+），**NMN = flagship 第一隻產品**。

## The spine — THE FIVE DECODES（每隻產品都印同一個 monospace block）
1. **The Molecule** — 成分 + form/salt + 每份精確劑量（form 係平牌藏品質嘅位，講出嚟就係 flex）
2. **The Purity** — assay 純度 %（NMN ≥99% β-form；omega-3 加 TOTOX + EPA/DHA mg），唔講「high quality」
3. **The Proof** — lot-matched COA（化驗報告）經 **/verify** 按 batch 攞 + named 第三方認證，唔講空洞「third-party tested」
4. **The Price** — price-per-dose + price-per-gram，計埋畀你睇，附「按配方成本定價、唔係 marketing」
5. **The Reasoning** — 一句 form rationale + 誠實 **Evidence Grade**（Strong / Moderate / Early）

## 三個難複製嘅護城河（COA-on-page 易抄，呢三個唔易）
- **The HYGIEIAS Transparency Standard** — 公開、versioned 嘅公司「憲法」(v1.0)，列明每隻 SKU 必達嘅自我規矩（Five Decodes + 無 proprietary blend + 全標示 + COA 出貨前公開）。把透明由 claim 變 auditable promise。
- **Evidence Grade badge** — 公開為自己嘅科學評級，弱嘅照標「Early」。配誠實句：「研究真實但仍早 —— 而我哋會話畀你聽幾早。」自我批判係 hype 對手結構上抄唔到嘅一招。
- **/verify** — 全 catalog 嘅 lot-number COA 查詢站，印喺每個標籤。

## Brand architecture — branded house
一個 masterbrand **HYGIEIAS** 蓋兩條 line（唔係 sub-brand）：**Foundations** + **Longevity**。共用：一套 COA/QA + /verify、一個 Decoded Panel 模板、一份 Transparency Standard、一套設計（navy `#1B2A4A` / 橙 `#D9531E` / mono；**一套 label kit —— 淨係 label 文字變，~15 SKU 近乎零設計成本，亦解決無攝影**）、一個 Stack builder。**線之分別 = 一條橙 accent stripe**（Foundations 純 navy；Longevity navy + 橙 stripe）。**NMN shock-isolation**：因 NMN 美國法規波動，獨立成一隻 SKU `L-NMN-300`，坐喺穩定 Foundations 收入之上。

## SKU 命名制
- **消費者名 = 成分 + Form/Salt + 每份劑量**（字面 = 前線透明；礦物必披露 elemental mg）
- **SKU code = [Line]-[Ingredient]-[Dose]**：F=Foundations、L=Longevity、S=Stack。例：`L-NMN-300`、`F-CREA-5000`、`F-MAG-200`、`S-HEALTHSPAN`。Lot = YYMM+run（`2406A`）。

## Roadmap
- **Phase 0 閘（硬 blocker）**：NMN 攞 company-specific NDIN（或簽持有 NDIN 嘅供應商）；**上線即出 /verify + Transparency Standard v1.0**；重新核 Amazon NMN 上架政策；標籤合規檢查（⚠️ critic 指原文「FDA Ingredient Traceability Code 2026-10」未證實/可能唔存在 → 待確認,勿照用）；起 claims-review gate。
- **Wave 1（launch）**：`L-NMN-300` NMN 300mg（flagship）+ `F-CREA-5000` Creatine + `F-MAG-200` Mag Glycinate + `F-D3K2-5000` D3+K2，配 `S-NAD` / `S-FOUND` / `S-HEALTHSPAN` bundle。NMN halo 做 launch 敘事；廣告投喺 Amazon-safe、無 NDI 負擔嘅 foundationals 做現金流 + 養 QA/COA 護城河。

## NMN flagship 經濟（一行）
原料 $0.18/g、landed COGS capsule $6.85（GM 80% @ $34.95）/ powder $20.50（GM 54% @ $44.95）；powder $/g ~$0.45 = 殺全場 hero number。（全模型見 [[NMN — Offer Architecture]]）

## Website IA 轉變
由 **NMN-only home** → **多產品 house**：Foundations / Longevity collection、可重用嘅 Decoded Panel PDP 模板、站內 /verify、Stack / Regimen builder。（見 [[Website IA (multi-product)]]）

## What changed from v2
- v2 = NMN 單一品牌；**v3 = 補充品公司，NMN = N 之 1**。
- 透明由「PDP feature」升做 **公司 OS**（Transparency Standard + Evidence Grade + /verify）。
- 加 foundationals 做信任/現金基座；**NMN 被 shock-isolate**（法規風險只傷一隻 SKU）。
- 品牌 story（Hygieia）只留 masterbrand 層；產品名一律字面化。
