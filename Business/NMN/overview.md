## 主軸

HYGIEIAS 補充品公司(branded house;Foundations + Longevity 兩條線,NMN = flagship)。知識分四大軸:

1. **供應商 Suppliers** — sourcing、報價、MOQ、lead time、樣品/COA、信譽
2. **產品 / 成分 Products** — 純度、規格、劑型、COA、成分知識(NMN、creatine、Mg glycinate、D3+K2…)
3. **市場 / 競品 Market** — 定價、競品、渠道(自家站/平台)、目標受眾
4. **法規 Regulatory** — 監管(美 FDA、NDI)、標籤、claims-safe、進口、合規

## 現況 takeaway（2026-06-28）

**v3 — 由 0 重建為補充品公司。** 定位由「NMN 單一品牌」→「**HYGIEIAS 補充品公司**,透明 = operating system」。Spine = **THE FIVE DECODES**(Molecule/Purity/Proof/Price/Reasoning)+ 三護城河(**Transparency Standard** 公司憲法・**Evidence Grade** 自評科學・**/verify** lot COA 站)。**7 份 v3 deliverable** 喺 `NMN/` root(全部對抗式 verify,claims-safe);v2 4 份凍結入 `NMN/V2/`,v1 喺 `NMN/V1/`。實數沿用:原料 $0.18/g、landed COGS(capsule $6.85 / powder $20.50)、≥99%、cGMP + NDIN。**下一步 = Phase 0 閘**(NDIN 鏈、/verify + Standard v1.0、Amazon 政策)+ sourcing + 履約定價。
