# NMN — overview

> Catalog-narrative(Karpathy overview):NMN 知識層嘅主軸、各 type 分佈、現況 takeaway、gap。隨內容累積更新。

## 主軸

NMN 補充品電商/零售。知識分四大軸:

1. **供應商 Suppliers** — sourcing、報價、MOQ、lead time、樣品/COA、信譽
2. **產品 / 成分 Products** — 純度、規格、劑型、COA、成分知識(NMN vs NR 等)
3. **市場 / 競品 Market** — 定價、競品、渠道(自家站/平台)、目標受眾
4. **法規 Regulatory** — 各地監管(美 FDA、歐、港、中)、標籤、進口、合規

## 現況 takeaway（2026-06-27）

**品牌 + offer + web 策略層已建（V2）。** 4 份 strategy deliverable(brand brief / brand story / offer architecture / PDP copy)喺 `NMN/` root,V1 存檔入 `NMN/V1/`(status: archived)。配套 web 已 build(home + capsule PDP + preview tool,repo `web/`,V1 喺 `web/V1/`)。實數已落:原料 $0.18/g、landed COGS(capsule $6.85 / powder $20.50)、≥99% 純度、cGMP + NDIN。**仲未做:** sourcing 落實(原料 NDIN 鏈確認)、Amazon 履約費鎖死、真 COA/lab 名、CAC 一層。下一步仍係 **sourcing + 履約定價**。
